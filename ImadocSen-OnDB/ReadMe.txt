##### readMe.txt ####
#
# Information on the content of ImadocSen-OnDB.
#
#####


This directory contains the following files and directories:

data/sentences: a directory containing the InkML files originally collected (one sub-directory per data collection session of a writer, and one file per sentence collected).

data/words: a directory containing InkML files corresponding to a manual segmentation of the phrases into their words (one sub-directory per data collection session of a writer, and one file per sentence collected).

dataset_infos.txt: a file containing information on the dataset (including descriptions of the data collection protocol, and of the InkML format used).

task1.txt: a file containing the distribution of the writers that we use, to divide them into the training set and the test set.
